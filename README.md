Python Slack Bot
=====================================

Install:

```
$pip install -r requirements.txt
```

Setting:
Override env.example.

```
$cp env.example .env
```

Start:

```
$python python-slack-bot.py .env
```

Usage:

```
naobot art: AIイラスト
naobot raw: AIイラスト（翻訳なし）
naobot talk: ChatGPT会話
naobot bokete: ChatGPT大喜利
```
